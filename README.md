# Defensive-Security-Project
This project demonstrates reviewing Windows and Apache Logs. This project we worked in a group and spit up the work I was tasked with reviewing the Apache Logs for a the simulated comapany **VSI**

# Objective
- Tracking what type of HTTP method was being used.
- Create a table showing the top 10 domains that refer to the VSI website
- Create a table showing the count of HTTP response codes
- Using splunk to compile this data

# Tools & Technologies
- Splunk
